## COMMANDS

### Start docker

#### Start from npm

`npm run docker`

#### List container

`docker ps -a`

#### Stop Container (hash = CONTAINER_ID )

`docker stop {hash}`

#### Remove Container (hash = CONTAINER_ID )

`docker rm {hash}`

#### Up Container (hash = CONTAINER_ID )

`docker-composer stop`

`docker-compose build`

`docker-composer up -d`

`docker-composer start`
